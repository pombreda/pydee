Global shortcuts
================

The default Pydee global shortcuts are the following:

- Maximizing current plugin / restoring original window layout:

  Ctrl+Shift+Alt+M

- Go to :doc:`editor`:
  
  Ctrl+Shift+E
  
- Go to :doc:`console`:

  Ctrl+Shift+C
  
- Go to :doc:`workspace`:

  Ctrl+Shift+W
  
- Go to :doc:`historylog`:

  Ctrl+Shift+H
  
- Go to :doc:`docviewer`:

  Ctrl+Shift+D
  
- Go to :doc:`extconsole`:

  Ctrl+Shift+X
