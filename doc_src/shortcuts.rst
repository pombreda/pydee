Global shortcuts
================

The default Pydee global shortcuts are the following:

- Editor:
  
  Ctrl+Maj+E
  
- Interactive console:

  Ctrl+Maj+C
  
- Workspace:

  Ctrl+Maj+W
  
- History:

  Ctrl+Maj+H
  
- Documentation viewer:

  Ctrl+Maj+D
  
- External console:

  Ctrl+Maj+X
