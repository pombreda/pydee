Global shortcutsÂ¶

The default Pydee global shortcuts are the following:

  • Maximizing current plugin / restoring original window layout:

    Ctrl+Shift+Alt+M

  • Go to Editor:

    Ctrl+Shift+E

  • Go to Interactive Console and Workspace:

    Ctrl+Shift+C

  • Go to Workspace:

    Ctrl+Shift+W

  • Go to History log:

    Ctrl+Shift+H

  • Go to Documentation viewer:

    Ctrl+Shift+D

  • Go to External Console:

    Ctrl+Shift+X
